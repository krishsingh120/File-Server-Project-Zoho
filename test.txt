Background (async):
  fileProcessorWorker
    → magic bytes check (fake extension detect)
    → MIME mismatch? → file delete + DB rejected
    → safe? → metadata extract → DB completed

  thumbnailWorker
    → sharp se 200x200 thumbnail banao
    → thumbnailPath DB mein save

  → Magic bytes check → MIME mismatch? file delete + rejected
  → Safe? → metadata save + completed
  → Image? → Thumbnail generate → thumbnailPath save
